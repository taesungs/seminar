세미나 안내

오는 5월 19일(월), AWS Edge Services Specialist SA들이 직접 발표 및 시연하는 AWS Best practices for web application security 세미나에 초대합니다.

AWS WAF, AWS Shield Advanced, Amazon CloudFront 등 AWS Edge Services를 활용하여 지능적으로 진화하는 웹 위협(봇, 크리덴셜 스터핑 및 크래킹, DDoS 등)으로부터 귀사의 워크로드를 보호할 수 있는 실제적인 방안을 소개합니다.

평소 AWS의 WAF나 Shield 등 경계 보안 (Perimeter Protection) 서비스를 사용하셨거나, 관심이 있으신 고객분들께서는 향후 각 보안 서비스의 발전 방향, 보안 모범 사례, 고객 사례 등을 참고하실 수 있습니다.

아무쪼록 동 행사를 통해 비즈니스 혁신 및 새로운 전략 수립에 도움이 될 수 있는 통찰력을 얻어 가시길 바랍니다.
